#  Regional Climate Data Analysis (2000–2021)

##  Overview
Basic exploratory data analysis of climate trends

##  Features
- Temperature trend analysis
- Rainfall variation study
- Humidity pattern analysis
- Correlation between variables

##  Insight
- Temperature shows slow increase
- Rainfall varies due to monsoon cycles
- Humidity is correlated with rainfall

##  Tools
Python, Pandas, Matplotlib, Seaborn

##  Run
pip install -r requirements.txt
python src/analysis.py
